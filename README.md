### Fibonacci Sequence Generator  

Generates the Fibonacci sequence up to a specified number of terms.  

**Description:**  
Takes an integer input `n` and prints the first `n` Fibonacci numbers, starting with 0 and 1.  
Uses a loop to calculate each term by summing the previous two numbers.  

**Example:**  
Input → 7  
Output → 0,1,1,2,3,5,8,
